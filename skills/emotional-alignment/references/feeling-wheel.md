# The Feeling Wheel — Disaggregated & Extended Reference

## How This Resource Works

The feeling wheel is organized as a strict four-ring hierarchy read **from the inside out**. Each step outward is determined by the prior step: a feeling at any ring is only reachable through its parent on the ring inside it.

**Traversal logic:** `Core Emotion → Secondary Feeling → Tertiary Feeling → Bounded Synonyms (Quaternary)`

**Structure:** 7 core emotions (Happy, Surprised, Bad, Fearful, Angry, Disgusted, Sad) → 41 secondary feelings (4–9 per core) → 82 tertiary feelings (exactly 2 per secondary) → 500 quaternary bounded synonyms (5–8 per tertiary; phrases permitted).

This resource is **bidirectional** — it serves identification and production equally — and it has a companion document, the **Phrase Sentiment Index** (phrase-sentiment-index.md), which maps ~1,900 English idioms and sayings onto these same paths.

### Purpose 1 — Identification (outside → inside)

Given any feeling word — a tertiary feeling or a quaternary bounded synonym — resolve it back through its parent feelings to one of the seven core emotions. This answers: *"I feel swamped — what am I actually feeling at the core?"* (swamped → Overwhelmed → Stressed → **Bad**.)

**Query recipe:**
1. Look the term up in the Reverse Lookup Index (Section 8), which is alphabetized and covers every tertiary and quaternary term.
2. Read off the full path. The bolded term in a path marks the tertiary feeling.
3. If the term carries a ⚠ flag, it appears in more than one path; disambiguate by asking which *parent feeling* fits the situation (see Duplicate-term notes below Section 8).

### Purpose 2 — Production (inside → outside)

Given an emotion at any ring, generate increasingly precise and varied language for that emotional territory. This answers: *"I need words for the specific kind of anger that comes from being deceived."* (Angry → Let down → Betrayed → double-crossed, backstabbed, sold out...)

**Query recipe:**
1. Enter the forward sections (Sections 1–7) at the core emotion.
2. Narrow by secondary feeling, then choose the tertiary feeling that best matches the intended shade.
3. Draw diction from that tertiary feeling's bounded synonym set — every word or phrase there is guaranteed consistent with the full path above it.
4. For idiomatic language on the same path, cross into the Phrase Sentiment Index's production section using the identical path notation.

### The Quaternary Ring: Bounded Synonyms

A bounded synonym is a synonym of a tertiary feeling filtered against the *entire path* above it: candidates contradicting the parent secondary feeling or core emotion are excluded.

Example: synonyms for *Aroused* are bounded by *Playful* and *Happy* — candidates contradicting playfulness or happiness (e.g., clinical or negative senses such as "agitated" or "provoked") are ruled out, leaving only language consistent with Happy → Playful → Aroused.

Because bounding is path-dependent, tertiary words appearing in **two different paths receive two different bounded synonym sets**. *Overwhelmed* under Bad → Stressed yields "swamped, snowed under, maxed out," while *Overwhelmed* under Fearful → Anxious yields "flooded, panicky, drowning in worry." Redundancy across quaternary sets is acceptable so long as each synonym adheres to the bounding of its own path.

### Reading Conventions

- Paths are always written with the arrow notation `Core → Secondary → Tertiary`; this notation is shared verbatim with the companion Phrase Sentiment Index, so paths are directly joinable across the two documents.
- In forward sections, each core emotion has a summary table plus an exhaustive "Full paths" list; either can be queried.
- The Reverse Lookup Index lists tertiary entries with the tertiary term **bolded** in the path, and quaternary entries pointing at their tertiary parent.
- ⚠ marks any term that appears in more than one location anywhere in the wheel.
