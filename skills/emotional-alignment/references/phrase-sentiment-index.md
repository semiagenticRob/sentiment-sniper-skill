# Phrase Sentiment Index — Complete (A–Z)

**Companion to:** *The Feeling Wheel — Disaggregated & Extended Reference* (feeling-wheel.md)
**Coverage:** Complete — 1871 English phrases classified against the feeling wheel's paths.

## Purpose

This index maps common English phrases, idioms, proverbs, and sayings onto the feeling wheel so that emotionally aligned language can be *identified* (phrase → feeling) and *produced* (feeling → phrase). It shares the wheel's exact path notation (`Core → Secondary → Tertiary`), so any path found here can be carried into the wheel document to retrieve its bounded synonyms, and vice versa.

## Source & Method

The phrase inventory is drawn from the master list of English phrases and sayings compiled by **Phrase Finder** (phrases.org.uk, "Over 2,100 English Phrases And Sayings"), accessed July 2026. Only phrase names are used; the site's explanatory content is not reproduced, and this index is unaffiliated with Phrase Finder. Sentiment classifications are original to this document, produced by an AI (Claude, Anthropic) as **inferential judgments about typical usage — not corpus-derived frequency data**.

## How to Read an Entry

Each row of the master table has four columns:

- **Phrase** — the phrase as listed in the source inventory.
- **Sentiment Path(s)** — one or more wheel paths. An unlabeled path is the default reading. A path prefixed with an italic label (e.g., *ironic:*, *subject:*, *recipient (grateful):*) is a **usage tag** — see Convention 4. Where multiple paths are listed, **the first is the predominant modern usage**.
- **Depth** — the ring of the *primary* path: Tertiary (most precise), Secondary, Core (broad emotional territory), or Neutral (no dominant sentiment). Shallower depth signals breadth, not lower quality.
- **Notes** — register cautions (ironic, dated, offensive history), wheel cross-references (where a phrase lands beside its own quaternary synonym), and secondary leans too weak to warrant a tagged path.

## Classification Conventions

1. **Connotation governs.** A phrase is placed by the feeling it *connotes in actual use*, never by surface denotation, and it is never sorted under a core emotion that contradicts that connotation. "As busy as a bee" denotes busyness but connotes cheerful vigor, so it sits under Surprised → Excited → Energetic — not the wheel's negative Bad → Busy.
2. **Deepest confident ring.** Each phrase is assigned the deepest feeling-wheel path (Core → Secondary → Tertiary) that can be claimed with confidence.
3. **Neutral.** Phrases with no dominant sentiment are tagged Neutral rather than forced onto a path.
4. **Usage tags (perspective and register).** By default a path describes the sentiment a **speaker/writer imparts by deploying the phrase**. Whenever perspective (*speaker*, *subject*, *recipient*) or register (*ironic*, *earnest*, *comic*, *negated*, *original vs. modern*, *lament*, *warning*) changes the feeling — including the core emotion — each usage carries its own labeled path, so every reading of the phrase remains in explicit alignment with the wheel. The first listed path is the predominant usage.
5. **Meta-entries excluded.** Site articles that are not phrases are excluded and listed at the end.

## Query Recipes

**Identification (phrase → feeling).** Find the phrase in the alphabetical master table (Section: Letter tables). Read the primary path for the default feeling; check usage tags if the perspective or register in your context differs. Example: encountering "cold shoulder" written from the recipient's side, take the *recipient* path (Fearful → Rejected → Excluded), not the giver's (Angry → Critical → Dismissive).

**Production (feeling → phrase).** Enter the "Production Direction" section at the desired core emotion for the candidate list, then verify each candidate's full path and register in the master table before use — the grouping keys off primary paths only, so a phrase listed under Happy may be gallows humor (e.g., "kick the bucket") unsuited to a sincere register. For single-word diction on the same path, carry the path into the feeling wheel's bounded synonym sets.

**Register matching.** To write in a specific tone (ironic, earnest, comic), prefer phrases whose *predominant* path already matches that register over phrases requiring their secondary tagged reading; the predominant reading is what most readers will hear first.

**Worked example.** Goal: express betrayal-flavored anger in a sincere register. Wheel: Angry → Let down → Betrayed → *double-crossed, backstabbed, deceived, sold out, cheated, played*. Index (same path): "double cross," "stab in the back," "sell down the river," "snake in the grass," "two-faced" — while "et tu, Brute" is flagged mock-reproach-predominant and is therefore reserved for comic registers.
